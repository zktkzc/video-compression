## 项目初始化

### 安装

```bash
$ pnpm install
```

### 开发

```bash
$ pnpm dev
```

### 编译

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
